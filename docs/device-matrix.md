# Device Matrix

SonimLab is currently developed and tested on a Verizon / Visible Sonim XP3800.

Different carrier variants may behave differently. Community reports are welcome.

| Device | Carrier / Network |  Camera | Tested | Notes                                                 |
| ------ | ----------------- | ------: | -----: | ----------------------------------------------------- |
| XP3800 | Verizon / Visible |      No |    Yes | Primary SonimLab development device                   |
| XP3800 | AT&T              | Unknown |     No | Community testing welcome                             |
| XP3800 | Sprint            | Unknown |     No | Known firmware differences; community testing welcome |
| XP3900 | Unknown           | Unknown |     No | Possible future investigation                         |

## Reported / Useful Differences to Track

* Boot image compatibility by carrier variant
* Hidden settings availability
* Notification behavior
* Hotspot / entitlement behavior
* Visual Voicemail behavior
* Carrier apps and MDM packages
* Hardware differences such as camera / non-camera models
